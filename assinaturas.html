Como o pagamento vai funcionar

Fluxo recomendado para não expor credenciais no navegador.

1. Usuário escolhe o planoO botão chama seu backend: /api/create-checkout.
2. Backend cria a cobrançaSeu servidor usa o token secreto do Mercado Pago.
3. Usuário paga no checkoutEle é redirecionado para pagar com PIX, cartão ou boleto.
4. Webhook confirmaSeu backend recebe a confirmação e libera Standard ou Plus no Firestore.
